# Death Head Hopper VR Bridge
**DeathHeadHopper VR Bridge** allows you to use **Death Head Hopper** in **VR** with **RepoXR**, adapting camera behavior and enabling ability slots through the Grip button.
This bridge is designed to make DeathHead gameplay fully usable and intuitive in VR without disrupting movement flow.

## Main Functionality
- Adjusts camera behavior during DeathHead mode  
- Enables ability slot usage via Grip combinations  
- Adds directional feedback for jumps and Charge ability  

## Controls Overview
### Grip Button Selection
The **Grip** button is automatically selected based on your **RepoXR Main Hand** setting.
- The mod uses the *opposite hand* of your Main Hand.
- This ensures the Grip used is always the one responsible for movement.
- You can manually force Right or Left Grip in the mod options if preferred.

### DeathHead Mode Controls
While in **DeathHead mode**:
- **Jump** → Use the normal directional joystick (default behavior)  
- **Move Camera** → Hold **Grip + Joystick**

This keeps standard movement intuitive while allowing full camera control when needed.

### Ability Usage (Grip Combos)
Abilities can be triggered using Grip combinations:
- **Grip + ResetHeight** → Activates **Charge** (Slot 1)  
- **Grip + Jump** → Activates **Slot 2 ability**

You can customize these bindings in the mod options.

**Note:**  
Slot 2 ability does not exist in vanilla DeathHeadHopper.  
It is available only if you are using **Last Chance**, an addon for DHHFix.

### Jump Direction & Visual Feedback
In the latest update:
- A raycast system has been added to the directional controller.  
- A red dot indicator shows the exact direction your head will travel.  

This applies to:
- Normal jumps  
- Charge ability  

This significantly improves aiming precision in VR.

## Compatibility
Tested on **Meta Quest**, but relies only on RepoXR input actions.  
Other headsets that map the same actions should work.
